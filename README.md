# grafika
Repository for computer graphics class.

Ürrepülés szimuláció mozgó aszteroida mezőben.


Irányítások:

W - előre mozgás
S - hátra mozgás
A - balra mozgás
D - jobbra mozgás

R - emelkedés
F - süllyedés

Q - balra forgás
E - Jobbra forgás

Kamera: 

I - előre mozgás
K - hátra mozgás
J - balra mozgás
L - jobbra mozgás


+ - fényerő növelés
- - fényerő csökkentés

T - help menü megnyitás/bezárás
